# Weather Application

A simple weather web application that fetches and displays live weather information using a public weather API.

## Features
- Displays current temperature and weather conditions
- Fetches real-time data using a REST API
- Clean and responsive user interface
- Easy to use and beginner-friendly

## Tech Stack
- HTML
- CSS
- JavaScript
- REST API

## Usage
1. Enter the name of a city
2. Click the search button
3. View the current weather details
